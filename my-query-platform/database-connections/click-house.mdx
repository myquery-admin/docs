### **Connection Method: Node.js Driver**

- **URL (Required):** The full HTTP/HTTPS endpoint.
  - _Example:_ 

    ```
    https://abcdef.us-east-1.aws.clickhouse.cloud:8443
    ```

    .
- **Username (Required):** Usually 

  ```
  default
  ```

  .
- **Password (Required):** Database password.
